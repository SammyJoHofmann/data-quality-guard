## Deutsch

### Verantwortlicher
Hoffmann Digital Solutions (HDS)
E-Mail: sh@hoffmanndigitalsolutions.de

### Welche Daten werden verarbeitet?
Die App verarbeitet ausschließlich Daten die bereits in Ihrer Atlassian-Instanz vorhanden sind:
- Jira: Ticket-Titel, Beschreibungen, Status, Assignees, Labels, Prioritäten, Erstellungs-/Änderungsdaten
- Confluence: Seitentitel, Seiteninhalte, Änderungsdaten

### Wo werden Daten gespeichert?
- Scan-Ergebnisse und Scores werden in der Forge SQL Datenbank gespeichert (gehostet von Atlassian in der EU/Region des Kunden)
- KEINE Daten werden auf externen Servern von HDS gespeichert
- KEINE Daten werden an Dritte weitergegeben

### KI-Verarbeitung (optional)
Wenn der Nutzer einen KI-API-Key eingibt und die KI-Analyse aktiviert:
- Ticket-Titel, Beschreibungen und Confluence-Seiteninhalte werden an den gewählten KI-Anbieter gesendet (Google Gemini, Anthropic Claude, oder OpenAI)
- Dies geschieht NUR wenn der Nutzer dies explizit aktiviert
- Der API-Key wird verschlüsselt in der Forge-Datenbank gespeichert
- Ohne KI-Aktivierung verlassen KEINE Daten die Atlassian-Plattform

### Datenlöschung
- Scan-Ergebnisse werden nach 90 Tagen automatisch gelöscht
- Bei Deinstallation der App werden alle Daten aus der Forge-Datenbank entfernt
- Der Nutzer kann jederzeit den API-Key löschen (Einstellungen → Key löschen)

### Rechtsgrundlage
Berechtigtes Interesse (Art. 6 Abs. 1 lit. f DSGVO) — Die Verarbeitung dient der Qualitätssicherung der Projektdaten des Nutzers.

## English

### Data Controller
Hoffmann Digital Solutions (HDS)
Email: sh@hoffmanndigitalsolutions.de

### What data is processed?
The app processes only data already present in your Atlassian instance:
- Jira: Ticket titles, descriptions, statuses, assignees, labels, priorities, creation/modification dates
- Confluence: Page titles, page contents, modification dates

### Where is data stored?
- Scan results and scores are stored in the Forge SQL database (hosted by Atlassian in the customer's region)
- NO data is stored on HDS external servers
- NO data is shared with third parties

### AI Processing (optional)
When the user enters an AI API key and enables AI analysis:
- Ticket titles, descriptions and Confluence page contents are sent to the selected AI provider
- This happens ONLY when explicitly enabled by the user
- Without AI activation, NO data leaves the Atlassian platform

### Data Deletion
- Scan results are automatically deleted after 90 days
- Upon app uninstallation, all data is removed from the Forge database
- Users can delete their API key at any time
